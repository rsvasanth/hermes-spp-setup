# Execution Plan for Hermes

**Instructions for the User:**

1. **Initialize Workspace**: Open your project in Hermes on the target computer.
2. **Setup Board**: Run the prompt in `setup_board.md`.
3. **Setup Roles**: Run the prompt in `setup_roles.md`.
4. **Index Knowledge**: Run `hermes curator archive ./docs/` (or wherever your HANDOVER.md is) to feed the project background to the Architect.
5. **Start Work**: Create a card for the first priority: "Analyze Purchase Receipt 'Not Synching' status and propose fix."
6. **Monitor Heartbeat**: Use the status bar to ensure agents are active and use the "Hallucination Gate" to verify completion logs.
